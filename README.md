# 🎨 生图生视频工具

一款集成式的 AI 图片生成工具，支持文生图和图生图功能，使用 nanobanana-2 模型（通过兔子 API）。

## ✨ 核心功能

- 📝 **文生图**：输入文本提示词快速生成图片
- 🖼️ **图生图**：上传本地图片或输入 URL，根据提示词生成新图片
- 📊 **任务队列**：支持批量提交任务，自动排队处理
- 📚 **历史记录**：完整的任务管理，支持查看、下载、重试、删除
- 🔄 **实时更新**：任务状态自动刷新，右上角显示当前任务数
- 🖱️ **图片预览**：点击图片放大查看

## 🚀 快速开始

### 方式一：一键启动（推荐）

#### Windows 用户

双击 **`启动工具.bat`** 即可启动

- 首次运行会自动安装依赖
- 浏览器会自动打开
- 关闭命令行窗口即可停止服务

#### Mac/Linux 用户

```bash
# 添加执行权限（首次运行）
chmod +x 启动工具.sh

# 启动
./启动工具.sh
```

### 方式二：命令行启动

```bash
# 1. 安装依赖（首次运行）
npm install

# 2. 配置 API Token（见下方说明）
# 编辑 .env 文件

# 3. 启动服务器
npm start

# 4. 浏览器访问
# http://localhost:3001
```

## ⚙️ 配置说明

### 1. 配置 API Token

启动前需要配置兔子 API 的 Token：

**编辑 `.env` 文件：**

```env
TUZI_API_TOKEN=sk-你的实际API密钥
TUZI_API_BASE_URL=https://api.tu-zi.com/v1
PORT=3001
POLL_INTERVAL=30000
POLL_TIMEOUT=300000
```

**重要参数说明：**

| 参数 | 说明 | 默认值 |
|-----|------|--------|
| `TUZI_API_TOKEN` | 兔子 API 密钥（**必填**） | - |
| `TUZI_API_BASE_URL` | API 基础 URL | `https://api.tu-zi.com/v1` |
| `PORT` | 服务器端口 | `3001` |
| `POLL_INTERVAL` | 轮询间隔（毫秒） | `30000`（30秒） |
| `POLL_TIMEOUT` | 任务超时时间（毫秒） | `300000`（5分钟） |

### 2. 获取 API Token

1. 访问兔子 API 平台
2. 注册账号并登录
3. 在控制台获取 API 密钥
4. 复制密钥到 `.env` 文件

## 📖 使用教程

### 文生图

1. 点击 **"文生图"** 选项卡
2. 在提示词输入框输入详细描述
3. 点击 **"生成图片"** 按钮
4. 自动跳转到历史记录，查看任务状态

**提示词示例：**
```
一个穿着蓝色连衣裙的亚洲女孩，
站在樱花树下，春风吹动樱花飘落，
明媚的阳光从树叶间洒下，
日式和风插画风格，温暖唯美，
高质量、细节丰富、光影渲染精致
```

### 图生图

1. 点击 **"图生图"** 选项卡
2. **上传本地图片** 或 **输入图片 URL**
3. 在提示词输入框描述想要的修改
4. 点击 **"生成图片"** 按钮

**支持的输入方式：**
- ✅ 上传本地图片（优先）
- ✅ 输入图片 URL（多个 URL 用换行分隔）

### 历史记录管理

在 **"历史记录"** 选项卡可以：

- 🔄 **自动刷新**：任务状态每 2 秒自动更新
- 📥 **下载图片**：点击"下载"按钮保存到本地
- 🔁 **重试失败任务**：失败的任务可以重新提交
- 🗑️ **删除记录**：单个删除或一键清空所有记录
- 🖼️ **放大预览**：点击图片查看大图

### 任务队列

右上角显示 **"当前任务: X"**，实时统计：

- 🟡 提交中的任务
- 🔵 处理中的任务

可以**连续提交多个任务**，系统会自动排队处理。

## ⚠️ 注意事项

### 安全性

1. **API Token 保护**
   - Token 存储在服务器 `.env` 文件中（**不要提交到 Git**）
   - 前端不会接触到 Token，更安全
   - 定期更换 Token

2. **本地部署**
   - 默认绑定 `localhost`，仅本机访问
   - 如需局域网访问，需修改服务器配置

### 使用限制

1. **API 配额**
   - 注意兔子 API 的调用次数限制
   - 每个任务通常耗时 30-60 秒
   - 避免短时间内提交过多任务

2. **图片大小**
   - 上传的本地图片会转为 base64
   - 建议单张图片不超过 5MB
   - 过大的图片可能导致请求超时

3. **浏览器兼容性**
   - 推荐使用现代浏览器（Chrome、Edge、Firefox）
   - IE 浏览器不支持

### 性能优化

1. **历史记录清理**
   - 定期清空不需要的历史记录
   - 过多记录会影响加载速度
   - 使用"一键清空"功能

2. **网络环境**
   - 确保网络稳定，避免任务提交失败
   - API 调用需要访问外网

## 🐛 常见问题

### 1. 启动失败

**问题：** 双击启动文件没有反应

**解决方案：**
```bash
# 检查 Node.js 是否安装
node --version

# 如果未安装，访问下载：
# https://nodejs.org/

# 手动安装依赖
npm install

# 查看错误日志
npm start
```

### 2. API Token 未设置

**问题：** 提示 "⚠️ 警告: TUZI_API_TOKEN 未设置"

**解决方案：**
1. 编辑 `.env` 文件
2. 将 `TUZI_API_TOKEN` 改为实际的 API 密钥
3. 重启服务器

### 3. 端口被占用

**问题：** 启动时提示端口 3001 已被占用

**解决方案：**
```env
# 编辑 .env 文件，修改端口号
PORT=8080
```

### 4. 任务一直显示"处理中"

**可能原因：**
- API 服务异常
- 网络连接问题
- 任务超时（默认 5 分钟）

**解决方案：**
1. 刷新历史记录
2. 检查网络连接
3. 查看任务是否已超时
4. 失败的任务可以点击"重试"

### 5. 图片无法下载

**解决方案：**
- 检查浏览器是否阻止下载
- 右键图片选择"另存为"
- 直接点击图片 URL 在新标签页打开
