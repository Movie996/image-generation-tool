# 快速开始指南

欢迎使用 **生图生视频工具**！按照下面的步骤，5 分钟内即可开始使用。

---

## ⚡ 开始前准备

你只需要准备一件事：**`.env` 配置文件**（包含你的 API Token）。

将 `.env` 文件放入项目文件夹根目录，内容示例：

```env
TUZI_API_TOKEN=sk-你的API密钥
PORT=3000
POLL_INTERVAL=10000
POLL_TIMEOUT=600000
```

> 参考 `.env.example` 文件查看所有可配置的参数。

---

## 🖥️ Windows 用户

**双击 `启动工具.bat`，其余全部自动完成。**

脚本会依次：

1. **检测 Node.js** — 未安装则用 `winget` 自动安装（需 Win10 1709+）；极少数不支持 winget 的系统会自动打开官网引导下载
2. **检测版本** — 自动校验 Node.js 是否 ≥ v20.6（版本过低会提示升级）
3. **安装依赖** — 首次运行自动执行 `npm install`，之后跳过
4. **启动服务器** — 在当前终端窗口启动，日志实时可见
5. **打开浏览器** — 2 秒后自动打开 http://localhost:3000

> **关闭终端窗口 = 停止服务器**，无需额外的停止脚本。

---

## 🍎 Mac / Linux 用户

**首次使用前**，需要在终端给启动脚本加上执行权限（只需做一次）：

```bash
chmod +x 启动工具.sh
```

之后每次直接运行：

```bash
./启动工具.sh
```

脚本同样会自动检查依赖、安装包、启动服务并打开浏览器。

## 🎨 使用方法

### 文生图

1. 打开浏览器访问 http://localhost:3000
2. 点击「文生图」选项卡
3. 输入详细的图片描述（中英文均可）
4. 点击 **生成图片**，看到成功提示后可继续输入下一个
5. 切换到「历史记录」查看生成进度和结果

**好的提示词示例：**
```
一位 25 岁亚洲女孩，长黑直发，浅蓝色棉麻连衣裙，
站在盛开的樱花树下，春风吹动花瓣飘落，
日式插画风格，色彩饱和度高，光影渲染细致，高质量
```

### 图生图

1. 点击「图生图」选项卡
2. 输入参考图片 URL（多张用换行分隔）
3. 输入修改描述，如：「转为动漫风格，背景改为蓝色天空」
4. 点击 **生成图片**

### 历史记录

| 操作 | 说明 |
|------|------|
| 👁 查看状态 | 实时显示 pending / completed / failed |
| 📥 下载 | 完成后点击下载到本地 |
| 🔁 重试 | 失败任务一键重新提交 |
| 🗑 删除 | 单条删除或一键清空 |

## 🔧 常见问题

**Q: 生成多久能出图？**
A: 通常 30 秒到 3 分钟，系统每 10 秒自动检查一次状态，无需手动刷新。

**Q: 可以同时提交多个任务吗？**
A: 可以，提交后按钮立即释放，连续提交不阻塞，所有任务在历史记录中追踪。

**Q: 端口 3000 被占用怎么办？**
A: 编辑 `.env` 文件，改 `PORT=8080`（或其他空闲端口），重启服务即可。

**Q: 任务一直失败？**
A: 检查网络是否能访问 `api.tu-zi.com`，确认 API Token 有效（以 `sk-` 开头）。
